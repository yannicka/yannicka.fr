Change brower bro'
Touches
  • Flèche haut OU D OU Espace : sauter
  • Flèche gauche OU Q : aller à gauche
  • Flèche droite OU D : aller à droite
  • Entrée : Valider le niveau une fois sur la porte avec la clé
  • Echap : retour au menu
But
Terminer tous les niveaux. Pour cela il faut prendre la clé dans chaque niveau puis aller à l'endroit où se situe la porte et appuyer sur entrée pour valider le niveau courant.
